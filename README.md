1、使用激活码完成激活

2、配置用于存储网址和笔记的本地excel文件路径

在搜索框中按如下方式输入， 然后回车

## 二、功能介绍
### 搜网址
命令格式：ww {关键字}
<img width="1074" height="431" alt="image" src="https://github.com/user-attachments/assets/9e58249e-ea7f-4270-bafb-a980c9f9bdb3" />

### 搜笔记
命令格式 ：nt {关键字}
<img width="1093" height="462" alt="image" src="https://github.com/user-attachments/assets/f38a14a4-f653-4d90-abf0-517b5cb326ef" />

### 搜全部
命令格式： {关键}

不带任何前缀，会同时搜索网址和笔记中的内容

<img width="1082" height="473" alt="image" src="https://github.com/user-attachments/assets/4b75d75b-ba51-4515-9260-7d0a875f1efa" />

### 搜copy历史记录
会把系统历史的copy记录展示出来

命令格式：hi 

注意命令后面一定要带空格

选中某个之后，敲击enter键，即可完成复制

### 快捷键
1）cmd + shift + 空格 ：唤起应用窗口并聚焦

2）enter： 选中某个下拉选项后，敲击enter，直接完成复制

3）enter：搜素框中完成关键词输入后，敲击enter，开始搜索
